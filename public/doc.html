图片上传 API

Bloret Image Host 提供了一个简单的 HTTP 接口用于上传图片。

接口地址

POST /api/upload

请求头 (Headers)

Content-Type: multipart/form-data

请求参数

参数名 类型 必填 说明
image File 是 要上传的图片文件 (支持 jpg, png, gif, webp)

响应示例 (成功)

{ "success": true, "message": "Upload successful", "data": { "url": "/img/1700000000000/a1b2c3d4e5f6...", "timestamp": 1700000000000, "md5": "a1b2c3d4e5f6...", "filename": "a1b2c3d4e5f6....jpg" } }

响应示例 (失败)

{ "success": false, "message": "No file uploaded" }

CURL 示例

curl -X POST -F "image=@/path/to/photo.jpg" http://localhost:3000/api/upload
